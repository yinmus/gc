## Установка

`Клон репозитория:`
```bash
git clone https://github.com/yinmus/gc.git
```

`Переход в директорию:`

```bash
cd gc
```

`Дать права на выполнение инсталлера:`

```bash
chmod +x gc_installer.sh
```

`Запустить инсталлер:`

```bash
./gc_installer.sh
```
